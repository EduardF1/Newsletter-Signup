# Newsletter-Signup
A Newsletter signup project built with NodeJS and Express that interacts with the Mailchimp API and is Live (deployed) on Heroku at `https://fast-hollows-21231.herokuapp.com/`.

- Dependencies: 
1) In the directory where the project is downloaded/cloned, run `npm init`.
2) External modules: run `npm i express` -> install Express, then run `npm i body-parser` -> install body-parser.

- Heroku:
1) Make an account at `https://heroku.com`.
2) Install the heroku CLI as specified at `https://devcenter.heroku.com/articles/getting-started-with-nodejs#set-up`.
3) Run `heroku login` to log in on the platform.
4) Check Git, Npm and Node versions, run `git --version`, `npm --version`, `node --version`.
5) Run `heroku create`.
6) Run `git add .` to start tracking your files with Git.
7) Run `git commit -m "yourCommitMessage"` to commit the files to the staging area.
8) Run `git push heroku master` to push the project files to heroku's server(s).
9) If everything is successful, a message containing a URL/link to the deployed app should appear in the terminal in blue (beware, it might take a couple of minutes for the application to be sucessfully deployed on the Heroku server(s)).
